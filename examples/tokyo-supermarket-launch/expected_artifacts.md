# Expected Artifacts

- `normalized_records.json`
- `index/manifest.json`
- `index/metadata.json`
- `population.json`
- `scenario_run.json`
- `scenario_run.jsonl`
- `aggregated_output.json`
- `aggregation_report.md`
- `demand_estimation.json`
- `demand_estimation_report.md`
- `evaluation_record.json`
- `evaluation_report.md`
